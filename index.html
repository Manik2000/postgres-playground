SQL Query Exercises

About

This page contains exercises to practice writing SQL queries. Click on each question to reveal the answer.

1. Select all columns from the 'customers' table.
SELECT * FROM customers;
2. Select distinct values from the 'country' column in the 'customers' table.
SELECT DISTINCT country FROM customers;
3. Select all customers from 'Germany' and order them by name.
SELECT * FROM customers
WHERE country = 'Germany'
ORDER BY customer_name;
