SQL Query Exercises

This page contains exercises to practice writing SQL queries. Click on each question to reveal the answer.
Set up the database according to the README instructions.

1. Select all columns from the 'games' table.
SELECT * FROM games;
2. Select distinct game ids of the played games from the 'player_games' table.
SELECT DISTINCT game_id FROM player_games;
3. Select the number of all records from the 'player_games' table.
SELECT count(*) FROM player_games;
4. Select top 5 duration of single gameplays and the ids and the dates for them.
Order by the duration, descending

SELECT duration, game_id, date 
FROM player_games
ORDER BY duration DESC
LIMIT 5;
5. Get all the players' names and the lengths of the names.
SELECT name, LENGTH(name) FROM player;
6. Get the minimal and maximal date.
SELECT MIN(date), MAX(date) FROM player_games;
7. Get the average score of the players.
SELECT AVG(score) FROM player_games;
8. Get the average gameplay time for each of the game ids.
Order by the average duration.
SELECT game_id, AVG(duration) FROM player_games GROUP BY game_id ORDER BY 2;
9. Get the games, average score and number of all gameplays between 2022-08-05 and 2023-09-25 for all games. Order by the number of games, descending.
SELECT game_id, AVG(score), count(*)
FROM player_games
WHERE date BETWEEN '2022-08-05' AND '2023-09-25'
GROUP BY game_id
ORDER BY 3 DESC;
